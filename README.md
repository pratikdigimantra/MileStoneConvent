# MileStone Convent Website

This repository contains a static website for MileStone Convent, including pages for about, academics, activities, admissions, virtual access, and the school homepage.

## Pages

- `index.html` - Homepage
- `about.html` - About the school
- `academics.html` - Academic programs and curriculum
- `activities.html` - Extracurricular activities
- `admission.html` - Admission process and requirements
- `virtual.html` - Virtual tour or online services

## Assets

- `assests/` - Site assets such as CSS, JavaScript, or other resources
- `images/` - Image files used by the website

## Usage

1. Open any `.html` file in a browser to view the page.
2. Edit content directly in the HTML files to update text, images, or page structure.

## Notes

- This is a static website, so no server or build process is required.
- Ensure media files are referenced correctly relative to the HTML files.
